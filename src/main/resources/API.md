### 微信端登陆
```
POST /wx/people/login
```
参数
```
account 教工号
password
types   1评委   2选手

```
返回  评委 1 选手 2
```
{
     "code": 0,
     "msg": "成功",
     "data": {
         "groups": 3,
         "types" : 1              //1 评委 ，2选手
         "id" : 34               //选手或评委id
         "activity_id": 66       //活动id
     }
 }
```

###评委打分按钮
```
POST /wx/people/mark
```
参数
```
playerId 选手主键
raterId  评委主键
score    总分

itemName ["演讲","现场"]   评分项名称
scoreItem [66，88]    评分项分数
groups 组别

```


###评委提前评分按钮
